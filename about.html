About HackShield

HackShield is a proof-of-concept project demonstrating a **fully deployed, secure cybersecurity dashboard**.

This application is built using a modern **Backend-as-a-Service (BaaS)** architecture, leveraging Firebase for both authentication and hosting.

Core Technologies:

  • Frontend: HTML5, CSS3, JavaScript (ES6+)
  • Authentication: Firebase Authentication (Google, Email)
  • Hosting: Firebase Hosting
  • Animation: particles.js
  • Data: Real-time data is currently *simulated* using JavaScript to demonstrate frontend visualization capabilities.
